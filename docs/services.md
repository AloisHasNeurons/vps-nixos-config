# Services Documentation

This directory contains detailed documentation for the services deployed on this VPS.

## Available Services

*   [AdGuard Home](./adguard.md) - Network-wide ad and tracker blocking.
*   [Homepage](./homepage.md) - A modern, fully static, fast, secure, fully proxied, highly customizable application dashboard.
*   [Vaultwarden](./vaultwarden.md) - Unofficial Bitwarden compatible server written in Rust.
*   [Nginx](./nginx.md) - High Performance Load Balancer, Web Server, & Reverse Proxy.
*   [WireGuard](./wireguard.md) - Fast, modern, secure VPN tunnel.
