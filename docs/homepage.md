# Homepage

**Homepage** is a modern, fully static, fast, secure, fully proxied, highly customizable application dashboard.

## Configuration

The configuration is located in [`modules/homepage.nix`](../modules/homepage.nix).

*   **Port**: `3001`

## Usage

Access the dashboard at `http://<server-ip>:3001`.

### Customization
You can customize the dashboard by editing the `settings` attribute in `modules/homepage.nix`.
Refer to the [Homepage documentation](https://gethomepage.dev/) for available options.
